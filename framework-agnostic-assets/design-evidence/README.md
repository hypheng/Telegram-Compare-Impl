# Design Evidence

这里存放设计评审证据和 handoff 记录。

建议放:

- 关键页面截图
- 没有 Figma 时的低保真草图、ASCII wireframe 或 annotated screenshot
- 设计评审结论
- ready-for-dev 清单
- Figma frame / node links
- 和实现偏差相关的说明

## 当前优先级

1. 为 `S1` 补 Figma backfill evidence
2. 为 `S2` 建立第一个完整的 Figma handoff evidence bundle
3. 具体清单见 `S1-S2-figma-delivery-checklist.md`

## 每个切片至少应留

- 切片名和日期
- Figma 文件链接
- 关键 frame / node links
- review 结论
- ready-for-dev 范围
- 实现偏差或待确认项
