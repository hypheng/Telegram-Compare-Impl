# Recommended Global Skills

全局 skills 负责通用能力，项目内 skills 负责这个仓库自己的 KMP / CJMP / parity workflow。

## 建议安装的全局 skills

| Skill | 用途 |
|---|---|
| `doc` | 输出设计、方案、交付文档 |
| `figma` | 读取设计文件、frame、节点和开发交付信息 |
| `figma-implement-design` | 从设计交付更稳定地生成实现任务 |
| `playwright` | 将来做 Android WebView / web 端 UI 取证和自动化 |
| `screenshot` | 采集 UI 证据 |
| `security-best-practices` | 收敛安全基线 |
| `security-threat-model` | 做聊天、账号、同步场景建模 |
