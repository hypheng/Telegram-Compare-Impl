# Parity Review Prompt

针对某个功能切片，检查 KMP 与 CJMP 的实现对齐情况。

要求:

1. 先读取 `framework-agnostic-assets/evaluation/parity-matrix.md`
2. 对照 `framework-agnostic-spec/` 中对应的需求与流程
3. 结合 `framework-agnostic-assets/evaluation/acceptance-reports/`、`ai-delivery-logs/` 和相关 issue
4. 分别评估 KMP / CJMP:
   - 功能完成度
   - 代码可维护性
   - 测试可达性
   - AI 交付顺滑度
