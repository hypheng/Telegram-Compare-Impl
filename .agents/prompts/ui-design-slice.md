# UI Design Slice Prompt

把某个 Telegram 功能切片补成可交付的 mobile UI/interaction 设计。

要求:

1. 先读取 `framework-agnostic-spec/requirements/`、`ux/`、`interface-design/`
2. 更新 `interface-design/screen-inventory.md`
3. 更新 `interface-design/interaction-states.md`
4. 如有视觉变化，更新 `interface-design/visual-system.md`
5. 在 `framework-agnostic-assets/design-evidence/` 记录 handoff 证据
6. 把本次 AI 设计执行过程记入 `framework-agnostic-assets/evaluation/ai-delivery-logs/`
7. 没有明确 UI 设计前，不直接进入实现
